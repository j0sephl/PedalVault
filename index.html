BOM Assistant - Generate BOMs with AI

Use AI tools like ChatGPT, Claude, or Gemini to generate BOMs from schematics and import them directly into PedalVault.

Step 1: Prepare Your Schematic

  • Take a clear photo or screenshot of your schematic
  • Ensure all component values and designators (R1, C2, etc.) are visible
  • For PDF schematics, you can upload them directly to most AI tools

Step 2: Use This Prompt Template

Copy this prompt and paste it with your schematic: 
Please analyze this guitar pedal schematic and create a Bill of Materials (BOM) in CSV format.

Format requirements:
- Column headers: "Name", "Quantity" 
- Use clear component names (e.g., "Resistor 10kΩ", "Capacitor 100nF")
- Include component values in the name
- Count total quantity needed for each unique component
- Do not include connectors, wires, or mechanical parts
- Combine duplicate components (e.g. if there are 3x 10kΩ resistors, show "Resistor 10kΩ, 3")

Example format:
Name,Quantity
Resistor 10kΩ,3
Capacitor 100nF,2
Op-Amp TL072,1

Please provide only the CSV data, no additional text or explanations.

Tips for Better Results

  • Be specific: Ask for "guitar pedal components" to avoid getting things like enclosures
  • Check the output: AI tools sometimes make mistakes - verify component values
  • Multiple tries: If results aren't good, try rephrasing your request
  • Combine similar parts: Ask the AI to merge similar components (e.g., all 10kΩ resistors)

About PedalVault

PedalVault is a guitar pedal parts inventory management system designed for DIY pedal builders and electronics enthusiasts.

Features

  • Track component inventory with quantities
  • Manage multiple pedal projects
  • Import/export BOMs (Bill of Materials)
  • NFC tag support for quick access
  • Progressive Web App (works offline)
  • Secure and privacy-focused (data stays local)
